Drie Lagen Model

Deze applicatie wordt op basis van drie lagen model geschreven. Hierbij ontstaat er een abstractie tussen verschillende lagen.

Front Layer

In deze laag zijn er zes functies:

  • user_input_init() In deze functie worden alle gebruikersinputs gelijk gemaakt aan 0.
  • lees_buffer() In deze functie worden alle ellementen (woorden voor de komma) uit een string opgesplitst en in een array van strings geplaatst.
  • gebruiker_input_herkennen() In deze functie worden ellementen uit array in een struct gezet.
  • kleur_herkennen() In deze functie wordt de kleurcode achterhaald. Input: string
    Output: int bijvoorbeeld: "wit" -> 255
  • tekst_stijl_herkennen() In deze functie wordt een string van tekst stijl omgezet in een int waarde. Input: string
    Output: int bijvoorbeeld: "nor" -> 0
  • commando_herkennen() In deze functie wordt een string van commando omgezet in een int waarde. Input: string
    Output: int bijvoorbeeld: "lijn" -> 0

Front Layer

Deze laag bevat een functie:

gebruiker_input_verwerking()

In deze functie wordt de gebruikersinput verwerkt. aan de hand hiervan wordt de juiste functies vanuit library aangeroepen.

Front Layer

Deze laag bevat alle hardware gerelateerde functies:

  • init_system_API() instellen van de microcontroller systeem
  • init_VGA_API() initialiseren van de VGA scherm
  • init_UART_API() initialisatie van UART
